## 📊 项目简介

本项目对上海二手房市场进行系统性分析，基于 **12 万 +** 真实房源数据，实现并对比了 **11 个回归模型**，包括传统统计模型、树模型集成方法和神经网络。通过深入的特征工程和数据质量检查，建立了可靠的房价预测系统。

## 🎯 核心成果

- **数据集**: 122,590 条训练数据，52,714 条测试数据
- **特征工程**: 31 个特征（27 个原始特征 + 8 个工程特征）
- **最佳模型**: XGBoost，R² = **0.8793**
- **关键发现**: 面积是房价的决定性因素（占特征重要性 68%）

## 📁 数据说明

### 数据来源
- **训练数据**: `dataset.csv` (122,590 条 × 27 特征)
- **测试数据**: `dataset_test.csv` (52,714 条 × 26 特征)
- **目标变量**: 价格（单位：万元）

| 朝向与交通 | 南、南北、近地铁 |
| 热度指标 | 多人关注 |

## 🔧 特征工程

### 新增特征（8 个）
1. **房龄** = 2026 - 建造年份
2. **房间密度** = 居室数 / 总面积
3. **厅室比** = 厅堂数 / 居室数
4. **总面积_平方** = 总面积²
5. **居室数_平方** = 居室数²
6. **房龄_面积交互** = 房龄 × 总面积
7. **是否次新** = (房龄 ≤ 5)
8. **房龄分段**、**面积分段**

### 关键技术
- **目标变量变换**: log(y) 解决右偏分布
- **分类特征编码**: Label Encoding 处理 15 个分类特征
- **数据清洗**: 移除 108 个无效样本（缺失值、0 值、负值）

## 📋 主要发现

### 1. 数据泄露陷阱
- **问题**: 单价 = 价格/面积，使用单价导致 R²虚高 0.08
- **教训**: 任何包含目标变量信息的特征都是作弊
- **解决**: 移除单价特征，建立特征审查清单

### 2. 面积是决定性因素
- 总面积_平方 (43.7%) + 总面积 (24.3%) = **68%**
- 符合房地产定价理论：面积是房价的首要决定因素

### 3. 位置因素重要
- 近地铁 (9.04%)
- 区域 (2.89%)
- 朝向 (1.04%)

### 4. log 变换的必要性
- 房价右偏分布（偏度 > 2）
- log 变换后更接近正态分布
- 模型更稳定，但还原时有误差
